# 🔧 問題修正 v2.2

## 已修正的問題

### 1. ✅ 機車避開高速公路

**問題**：機車模式會走國道高速公路

**修正**：
- 加入 `avoidHighways: true` 參數
- 機車模式現在會避開高速公路
- 只走省道和一般道路

**程式碼變更**：
```javascript
case 'TWO_WHEELER':
    travelMode = google.maps.TravelMode.DRIVING;
    avoidHighways = true; // 機車避開高速公路
    break;
```

---

### 2. ✅ 騎行時數正確計算

**問題**：騎行時數沒有根據實際路線更新

**修正**：
- 使用 Google Directions API 回傳的實際時間
- 不再使用簡單的距離÷速度計算
- 考慮路況、道路類型等因素

**程式碼變更**：
```javascript
let totalDuration = 0;
for (let i = 0; i < route.legs.length; i++) {
    totalDuration += route.legs[i].duration.value;
}
const durationHours = (totalDuration / 3600).toFixed(1);
```

---

### 3. ✅ 地點搜尋先標記再確認

**問題**：搜尋地點後直接加入路線，無法取消

**修正**：
- 搜尋後先在地圖上顯示藍色標記
- 彈出資訊視窗詢問是否加入
- 提供「加入路線」和「取消」按鈕
- 使用者可以決定是否加入

**使用流程**：
```
1. 搜尋「台北101」
2. 地圖顯示藍色標記（跳動動畫）
3. 彈出視窗：
   [加入路線] [取消]
4. 點擊「加入路線」才會加入
```

---

### 4. ✅ 天氣 API 設定

**問題**：無法取得天氣資訊

**原因**：未設定 OpenWeather API Key

**修正**：
- 在 `.env` 檔案加入 `OPENWEATHER_API_KEY` 欄位
- 提供清楚的設定說明

**設定步驟**：
```
1. 前往 https://openweathermap.org/api
2. 註冊免費帳號
3. 取得 API Key
4. 編輯 .env 檔案：
   OPENWEATHER_API_KEY=你的API_Key
5. 重新啟動伺服器
```

---

### 5. ✅ 每日行程規劃功能

**問題**：無法分天數規劃行程

**新增功能**：
- 新增「每日行程」面板
- 可以為每一天選擇終點
- 顯示每日起點和終點
- 方便規劃多日行程

**使用方式**：
```
1. 新增至少 2 個路線點
2. 點擊「規劃每日行程」按鈕
3. 為每一天選擇終點：
   - 第 1 天終點：台中
   - 第 2 天終點：台南
   - 第 3 天終點：高雄
4. 點擊「確定」
5. 顯示每日行程規劃
```

---

### 6. ✅ 附近搜尋改為點擊路線點

**問題**：附近搜尋只能搜尋最後一個路線點

**修正**：
- 先選擇搜尋類型（住宿、加油站等）
- 點擊路線點旁的 🔍 按鈕
- 搜尋該路線點附近的地點
- 更靈活的搜尋方式

**使用流程**：
```
1. 點擊「🏨 住宿」按鈕（選擇類型）
2. 在路線點列表中，點擊想搜尋的路線點旁的 🔍
3. 顯示該路線點附近的住宿
4. 點擊標記查看詳細資訊
```

## 技術改進

### 路線計算優化
```javascript
// 之前：簡單計算
estimatedHours = distance / 50

// 現在：使用實際時間
totalDuration = route.legs[i].duration.value
durationHours = totalDuration / 3600
```

### 機車路線優化
```javascript
// 之前：與汽車相同
travelMode: google.maps.TravelMode.DRIVING

// 現在：避開高速公路
travelMode: google.maps.TravelMode.DRIVING,
avoidHighways: true
```


## 已知限制

### 1. Google Maps API 限制
- 機車模式仍使用 DRIVING（Google 沒有獨立機車模式）
- 透過 `avoidHighways` 避開高速公路
- 路線可能不是最佳機車路線

### 2. 天氣 API 限制
- 免費帳號每月 1,000 次呼叫
- 超過額度會無法取得天氣
- 需要自行註冊 API Key

### 3. 每日行程規劃
- 目前需要手動選擇每日終點
- 未來可加入自動分配功能
- 未來可加入每日距離限制

## 常見問題

### Q: 機車還是走高速公路？
**A:** 
- 清除瀏覽器快取
- 確認已重新啟動伺服器
- 檢查 Console 是否有錯誤

### Q: 騎行時數沒變？
**A:** 
- 重新計算路線（切換交通工具）
- 確認路線已正確繪製
- 檢查是否有錯誤訊息

### Q: 搜尋後沒有彈出視窗？
**A:** 
- 檢查瀏覽器是否阻擋彈出視窗
- 查看 Console 錯誤訊息
- 確認 Places API 已啟用

### Q: 無法取得天氣？
**A:** 
- 確認已設定 OPENWEATHER_API_KEY
- 檢查 API Key 是否有效
- 確認未超過免費額度

### Q: 每日行程按鈕沒出現？
**A:** 
- 確認已新增至少 2 個路線點
- 重新整理頁面
- 檢查 Console 錯誤
