## 🎯 使用情境範例

### 情境 1：規劃台灣環島七日遊

#### 步驟 1：建立行程
```
1. 開啟網站 http://localhost:5000
2. 填寫行程名稱：「台灣環島七日遊」
3. 填寫描述：「順時針環島，慢遊台灣」
```

#### 步驟 2：新增路線點
```
點擊地圖新增以下地點：
1. 台北市 (25.0330, 121.5654)
2. 宜蘭市 (24.7021, 121.7378)
3. 花蓮市 (23.9871, 121.6015)
4. 台東市 (22.7583, 121.1444)
5. 墾丁 (21.9458, 120.7967)
6. 高雄市 (22.6273, 120.3014)
7. 台南市 (22.9998, 120.2269)
8. 台中市 (24.1477, 120.6736)
9. 新竹市 (24.8138, 120.9675)
10. 回到台北
```

#### 步驟 3：查看統計
```
系統自動計算：
📊 總距離：約 1,200 km
📅 預計天數：6-7 天
⏰ 騎行時數：約 24 小時
⛽ 油耗費用：約 NT$ 1,440
💰 總費用：約 NT$ 7,940
  - 油費：NT$ 1,440
  - 住宿：NT$ 6,000 (6晚)
  - 餐費：NT$ 3,500 (7天)
```

#### 步驟 4：搜尋住宿
```
在每個停留點搜尋住宿：
1. 宜蘭：點擊 [🏨] 搜尋附近住宿
2. 花蓮：點擊 [🏨] 搜尋附近住宿
3. 台東：點擊 [🏨] 搜尋附近住宿
... 依此類推
```

#### 步驟 5：查看天氣
```
點擊 [查看天氣] 按鈕
查看未來天氣預報：
☀️ 明天：晴天 28°C
🌤️ 後天：多雲 26°C
```

#### 步驟 6：儲存與分享
```
1. 點擊 [💾 儲存行程]
2. 點擊 [🔗 分享] 取得分享連結
3. 點擊 [📄 匯出PDF] 下載行程表
```

## 🎨 功能展示

### 1. 地圖互動

```
操作方式：
1. 點擊地圖 → 新增路線點
2. 拖曳地圖 → 移動視角
3. 滾輪縮放 → 放大/縮小
4. 點擊標記 → 查看資訊
```

### 2. 路線規劃

```
自動功能：
✅ 自動連接路線點
✅ 自動計算最佳路線
✅ 自動顯示距離
✅ 自動估算時間
```

### 3. 地點搜尋

```
搜尋類型：
🏨 住宿：飯店、民宿、青年旅館
⛽ 加油站：中油、台塑、全國
🍴 餐廳：小吃、餐廳、咖啡廳
📷 景點：觀光景點、博物館、公園

搜尋範圍：
📍 以最後路線點為中心
📏 半徑 10 公里
📊 顯示前 10 個結果
```

### 4. 天氣預報

```
顯示資訊：
🌡️ 溫度：攝氏度
☁️ 天氣：晴天/多雲/雨天
💨 風速：公里/小時
💧 濕度：百分比

預報範圍：
📅 未來 2 天
⏰ 每 3 小時更新
```

### 5. 費用估算

```
計算項目：
⛽ 油費 = 距離 ÷ 25 km/L × NT$30/L
🏨 住宿 = (天數-1) × NT$1,000/晚
🍴 餐費 = 天數 × NT$500/天

範例：
500 km, 3 天
⛽ 油費：500÷25×30 = NT$600
🏨 住宿：2×1000 = NT$2,000
🍴 餐費：3×500 = NT$1,500
💰 總計：NT$4,100
```

### 6. 分享功能

```
分享方式：
1. 產生唯一連結
   https://yoursite.com/share/abc123xyz

2. 自動複製到剪貼簿
   ✅ 連結已複製！

3. 分享給朋友
   📱 LINE、Facebook、Email

4. 掃描 QR Code
   📱 手機掃描即可查看
```

### 7. PDF 匯出

```
PDF 內容：
📄 第 1 頁：
   - 行程名稱
   - 行程描述
   - 統計資訊
   - 路線點列表

📄 第 2 頁：
   - QR Code
   - 分享連結
   - 建立日期

檔案格式：
📁 檔名：行程名稱.pdf
📏 大小：約 100-200 KB
📱 支援手機查看
```

## 💡 使用技巧

### 技巧 1：快速規劃

```
1. 使用地圖搜尋功能
   - 在搜尋框輸入地點名稱
   - 點擊搜尋結果
   - 自動新增到路線

2. 使用右鍵選單（未來功能）
   - 右鍵點擊地圖
   - 選擇「新增路線點」
   - 選擇「搜尋附近」
```

### 技巧 2：優化路線

```
1. 調整路線點順序
   - 拖曳路線點重新排序
   - 系統自動重新計算

2. 刪除不需要的點
   - 點擊路線點的 [🗑️] 按鈕
   - 確認刪除
```

### 技巧 3：預算控制

```
1. 自訂費率
   - 編輯 config.py
   - 修改預設值
   - 重新啟動伺服器

2. 查看費用明細
   - 油費明細
   - 住宿明細
   - 餐費明細
```

### 技巧 4：天氣規劃

```
1. 查看多日天氣
   - 點擊「查看天氣」
   - 查看未來預報
   - 調整出發日期

2. 避開壞天氣
   - 如果預報下雨
   - 調整行程順序
   - 或延後出發
```
