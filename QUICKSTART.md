# 🚀 快速開始指南

## 一鍵啟動（推薦）

```bash
./start.sh
```

這個腳本會自動：
- ✅ 檢查 Python 環境
- ✅ 安裝依賴套件
- ✅ 初始化資料庫
- ✅ 啟動伺服器

## 手動啟動

### 步驟 1：安裝依賴

```bash
pip install -r requirements.txt
```

### 步驟 2：設定環境變數

複製 `.env.example` 為 `.env`：

```bash
cp .env.example .env
```

編輯 `.env` 並填入你的 API Keys：

```env
SECRET_KEY=your-secret-key
GOOGLE_MAPS_API_KEY=你的_Google_Maps_API_Key
OPENWEATHER_API_KEY=你的_OpenWeather_API_Key（選用）
```

### 步驟 3：初始化資料庫

```bash
python init_db.py
```

### 步驟 4：啟動伺服器

```bash
python app.py
```

### 步驟 5：開啟瀏覽器

訪問 `http://localhost:5000`

## 📱 手機訪問

1. 確保手機和電腦在同一個 Wi-Fi 網路
2. 查詢電腦的 IP 位址：
   ```bash
   # macOS/Linux
   ifconfig | grep "inet "
   
   # Windows
   ipconfig
   ```
3. 在手機瀏覽器輸入：`http://你的電腦IP:5000`

## 🎯 第一次使用

### 1. 建立你的第一個行程

1. **新增路線點**
   - 在地圖上點擊想去的地方
   - 系統會自動編號並連接路線

2. **查看統計資訊**
   - 總距離
   - 預計天數
   - 騎行時數
   - 油耗費用
   - 總費用

3. **搜尋附近設施**
   - 點擊「住宿」搜尋飯店
   - 點擊「加油站」找加油站
   - 點擊「餐廳」找美食
   - 點擊「景點」找觀光景點

4. **儲存行程**
   - 填寫行程名稱（例如：台灣環島七日遊）
   - 填寫行程描述（選填）
   - 點擊「儲存行程」

### 2. 使用進階功能

#### ☁️ 查看天氣
```
1. 新增路線點
2. 點擊「查看天氣」
3. 查看未來天氣預報
```

#### 🔗 分享給朋友
```
1. 儲存行程後
2. 點擊「分享」按鈕
3. 連結自動複製
4. 傳送給朋友
```

#### 📄 匯出 PDF
```
1. 儲存行程後
2. 點擊「匯出 PDF」
3. 下載行程表
```

## 🔧 常見問題

### Q: 地圖無法顯示？
**A:** 
1. 檢查 Google Maps API Key 是否正確
2. 確認已啟用 Maps JavaScript API
3. 按 F12 查看 Console 錯誤訊息

### Q: 天氣功能無法使用？
**A:** 
1. 確認已設定 OpenWeatherMap API Key
2. 檢查 API Key 是否有效
3. 免費帳號每月限制 1000 次呼叫

### Q: 如何修改預設費用？
**A:** 
編輯 `config.py` 檔案：
```python
FUEL_CONSUMPTION = 25  # 改成你的機車油耗
FUEL_PRICE = 30  # 改成當前油價
DAILY_ACCOMMODATION = 1000  # 改成你的住宿預算
DAILY_MEAL = 500  # 改成你的餐費預算
```

### Q: 可以離線使用嗎？
**A:** 
目前需要網路連線使用 Google Maps。離線地圖功能正在開發中。

### Q: 如何備份我的行程？
**A:** 
1. 方法一：匯出 PDF 保存
2. 方法二：複製 `trips.db` 檔案備份
3. 方法三：使用分享連結（雲端儲存）
