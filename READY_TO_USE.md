## 💡 快速測試

### 測試機車路線

```
1. 選擇「🏍️ 機車」
2. 搜尋「台北車站」→ 點擊「加入路線」
3. 搜尋「台中火車站」→ 點擊「加入路線」
4. 查看路線（應該不走高速公路）
5. 查看騎行時數（應該顯示實際時間）
```

### 測試搜尋功能

```
1. 搜尋「日月潭」
2. 看到藍色標記（跳動）
3. 彈出視窗：[加入路線] [取消]
4. 點擊「加入路線」或「取消」
```

### 測試每日行程

```
1. 新增 3 個路線點
2. 點擊「規劃每日行程」
3. 選擇每日終點
4. 查看每日行程卡片
```

### 測試附近搜尋

```
1. 點擊「🏨 住宿」（選擇類型）
2. 在路線點旁點擊 🔍
3. 查看附近住宿
```

## ⚙️ 選用設定

### 天氣功能（選用）

如果想使用天氣功能：

```
1. 前往 https://openweathermap.org/api
2. 註冊免費帳號
3. 取得 API Key
4. 編輯 .env 檔案：
   OPENWEATHER_API_KEY=你的API_Key
5. 重新啟動伺服器
```

## 📱 手機使用

### 在手機上訪問

```
1. 確保手機和電腦在同一 Wi-Fi
2. 查詢電腦 IP：
   ifconfig | grep "inet " (Mac/Linux)
   ipconfig (Windows)
3. 在手機瀏覽器輸入：
   http://你的電腦IP:5000
```

## 🆘 需要幫助？

### 常見問題

**Q: 機車還是走高速公路？**
A: 清除瀏覽器快取，重新啟動伺服器

**Q: 搜尋沒有彈出視窗？**
A: 檢查瀏覽器是否阻擋彈出視窗

**Q: 天氣無法顯示？**
A: 需要設定 OPENWEATHER_API_KEY

**Q: 每日行程按鈕沒出現？**
A: 需要至少 2 個路線點
