## 搜尋功能對比

### v2.0 及之前

**只能透過地圖操作**
```
方式 1：點擊地圖
- 需要知道地點位置
- 需要手動縮放地圖
- 不知道確切座標

方式 2：附近搜尋
- 只能搜尋特定類型
- 需要先有路線點
- 範圍限制 10 km
```

### v2.1 新增

**可以直接搜尋地點**
```
方式 1：地點搜尋
✅ 輸入地點名稱
✅ 自動完成建議
✅ 精確找到地點
✅ 自動新增到路線

方式 2：點擊地圖（保留）
✅ 快速新增中途點
✅ 自訂位置

方式 3：附近搜尋（保留）
✅ 找住宿、加油站
✅ 找餐廳、景點
```

## 使用流程對比

### v2.0 規劃流程

```
1. 手動縮放地圖找到起點
2. 點擊地圖新增起點
3. 手動縮放地圖找到下一個點
4. 點擊地圖新增
5. 重複步驟 3-4
6. 查看統計資訊
7. 儲存行程

問題：
❌ 需要知道地點位置
❌ 需要多次縮放地圖
❌ 費時費力
```

### v2.1 規劃流程

```
1. 選擇交通工具（🏍️ 機車）
2. 搜尋「台北車站」→ Enter
3. 搜尋「台中火車站」→ Enter
4. 搜尋「高雄火車站」→ Enter
5. 查看統計資訊
6. 儲存行程

優點：
✅ 快速新增地點
✅ 不需要知道位置
✅ 自動完成建議
✅ 省時省力
```
