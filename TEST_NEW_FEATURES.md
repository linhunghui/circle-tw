## 常見問題排除

### 問題 1：搜尋沒反應
```
解決方法：
1. 檢查 Console 錯誤
2. 確認 Places API 已啟用
3. 檢查 API Key 配額
4. 重新整理頁面
```

### 問題 2：自動完成不出現
```
解決方法：
1. 確認網路連線
2. 檢查 API Key 權限
3. 清除瀏覽器快取
4. 嘗試無痕模式
```

### 問題 3：交通工具切換無效
```
解決方法：
1. 確認已有至少 2 個路線點
2. 檢查 Console 錯誤
3. 重新新增路線點
4. 重新整理頁面
```

### 問題 4：手機版搜尋框太小
```
解決方法：
1. 檢查 CSS 是否載入
2. 清除手機瀏覽器快取
3. 嘗試橫向模式
```
