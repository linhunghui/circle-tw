## 🎊 開始使用

```bash
# 1. 安裝依賴
pip install -r requirements.txt

# 2. 設定 API Keys
cp .env.example .env
# 編輯 .env 填入你的 API Keys

# 3. 初始化資料庫
python init_db.py

# 4. 啟動伺服器
python app.py

# 5. 開啟瀏覽器
# http://localhost:5000
```
