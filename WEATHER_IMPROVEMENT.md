## 🔧 改善內容

### 1. **修復地點名稱顯示**

**問題**：天氣預報顯示的城市名稱不正確或是英文

**原因**：
- OpenWeather API 回傳的城市名稱可能是英文（例如：Taipei, Kaohsiung）
- 或是不正確的中文翻譯
- 無法準確對應台灣的地點名稱

**解決方案**：
- 前端傳送路線點的名稱給後端
- 後端優先使用前端傳來的地點名稱
- 確保顯示的是使用者新增路線點時的名稱

#### 程式碼變更：

**前端（static/js/app.js）**：
```javascript
// 傳送地點名稱給後端
body: JSON.stringify({
    lat: waypoint.lat,
    lng: waypoint.lng,
    name: waypoint.name  // 新增：傳送地點名稱
})
```

**後端（app.py）**：
```python
# 接收地點名稱
location_name = data.get('name', '')

# 優先使用前端傳來的地點名稱
city_name = location_name if location_name else weather_data['city']['name']

return jsonify({
    'city': city_name,
    'forecasts': forecasts
})
```

## 📊 天氣資訊說明

### OpenWeather API 提供的資料

#### 溫度：
- **當前溫度**：該時段的溫度
- **單位**：攝氏度（°C）
- **更新頻率**：每 3 小時

#### 天氣描述：
OpenWeather 支援繁體中文（`lang=zh_tw`），常見描述：
- ☀️ 晴天
- ⛅ 多雲
- ☁️ 陰天
- 🌧️ 小雨、中雨、大雨
- ⛈️ 雷陣雨
- 🌫️ 霧
- 🌪️ 強風

#### 風速：
- **單位**：km/h（公里/小時）
- **轉換**：API 回傳 m/s，程式自動轉換為 km/h
- **計算**：`wind_speed * 3.6`

#### 濕度：
- **單位**：百分比（%）
- **範圍**：0-100%
- **說明**：
  - 30-50%：舒適
  - 50-70%：稍微潮濕
  - 70%+：潮濕

## 🔍 常見問題

### Q1: 為什麼天氣預報不準確？

**A**: 天氣預報的準確度受多種因素影響：

1. **資料來源**：OpenWeather 使用全球氣象資料
2. **台灣地形**：山區、海岸地形複雜，預報較困難
3. **更新頻率**：每 3 小時更新一次
4. **預報時間**：越遠的時間越不準確

**建議**：
- 出發前查看中央氣象局的預報
- 天氣預報僅供參考
- 隨時注意天氣變化

### Q2: 為什麼有些地點顯示英文名稱？

**A**: 可能的原因：

1. **路線點名稱**：如果路線點是點擊地圖新增的，名稱會是座標
2. **API 限制**：OpenWeather 可能沒有該地點的中文名稱

**解決方法**：
- 使用搜尋功能新增路線點（會有正確的中文名稱）
- 或手動編輯路線點名稱（未來功能）

### Q3: 為什麼只顯示 4 個時段？

**A**: 設計考量：

1. **資訊視窗大小**：避免內容過多
2. **實用性**：24 小時內的預報最準確
3. **載入速度**：減少資料量

**如果需要更長時間的預報**：
- 可以查看不同路線點的天氣
- 或使用中央氣象局網站

### Q4: 天氣圖示代表什麼？

**A**: OpenWeather 天氣圖示說明：

| 圖示 | 代碼 | 說明 |
|------|------|------|
| ☀️ | 01d | 晴天（白天）|
| 🌙 | 01n | 晴天（夜晚）|
| ⛅ | 02d/02n | 少雲 |
| ☁️ | 03d/03n | 多雲 |
| ☁️ | 04d/04n | 陰天 |
| 🌧️ | 09d/09n | 陣雨 |
| 🌧️ | 10d/10n | 雨 |
| ⛈️ | 11d/11n | 雷雨 |
| 🌨️ | 13d/13n | 雪 |
| 🌫️ | 50d/50n | 霧 |

## 🌡️ 騎車天氣建議

### 適合騎車的天氣：
- ✅ 溫度：15-28°C
- ✅ 天氣：晴天、多雲
- ✅ 風速：< 20 km/h
- ✅ 濕度：< 70%

### 不適合騎車的天氣：
- ❌ 下雨（路面濕滑）
- ❌ 強風（> 30 km/h）
- ❌ 高溫（> 35°C，容易中暑）
- ❌ 低溫（< 10°C，容易失溫）
- ❌ 霧（能見度低）

### 騎車天氣注意事項：

#### 🌧️ 雨天：
- 攜帶雨衣
- 降低速度
- 增加跟車距離
- 注意路面積水

#### ☀️ 晴天高溫：
- 防曬（防曬乳、袖套）
- 補充水分
- 避開中午時段（11:00-14:00）
- 穿著透氣衣物

#### 💨 強風：
- 降低速度
- 注意側風
- 避開空曠路段
- 雙手緊握把手

#### 🌫️ 霧天：
- 開啟車燈
- 降低速度
- 保持安全距離
- 必要時停車等待
