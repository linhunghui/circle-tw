# 🎉 更新說明 v2.1

## 新增功能

### 1. 🔍 地點搜尋功能

現在你可以直接在搜尋框輸入地點名稱，系統會自動搜尋並新增到路線中！

#### 使用方式：
```
1. 在「地點搜尋」面板找到搜尋框
2. 輸入地點名稱（例如：台北101、日月潭、墾丁大街）
3. 點擊「搜尋並新增」按鈕或按 Enter
4. 系統會自動：
   - 找到該地點
   - 新增到路線中
   - 移動地圖到該位置
```

#### 功能特色：
- ✅ 自動完成建議（輸入時會顯示建議地點）
- ✅ 限制台灣地區（搜尋結果更精確）
- ✅ 支援各種地點類型（景點、餐廳、飯店等）
- ✅ 按 Enter 快速搜尋

#### 範例：
```
搜尋「台北車站」→ 自動新增到路線
搜尋「阿里山」→ 自動新增到路線
搜尋「墾丁國家公園」→ 自動新增到路線
```

### 2. 🚗 交通工具選擇

現在可以選擇不同的交通工具來規劃路線！

#### 可選擇的交通工具：
- 🚗 **汽車** - 一般道路路線
- 🏍️ **機車** - 預設選項，適合環島
- 🚴 **自行車** - 自行車專用道優先
- 🚶 **步行** - 步行路線

#### 使用方式：
```
1. 在「行程資訊」面板找到「交通工具」下拉選單
2. 選擇你的交通工具
3. 系統會自動重新計算路線
```

#### 差異說明：

**🚗 汽車模式**
- 使用一般道路
- 可走高速公路
- 速度較快

**🏍️ 機車模式**
- 使用一般道路（與汽車相同）
- 適合環島規劃
- 預設選項

**🚴 自行車模式**
- 優先使用自行車道
- 避開高速公路
- 路線較安全

**🚶 步行模式**
- 使用人行道
- 距離較短
- 適合市區觀光

## 已知限制

### Google Maps API 限制

1. **機車模式**
   - Google Maps 沒有獨立的機車模式
   - 目前使用「汽車」模式代替
   - 路線規劃與汽車相同

2. **自行車模式**
   - 部分地區可能沒有自行車道資料
   - 會退回使用一般道路

3. **步行模式**
   - 長距離步行可能無法計算
   - 建議用於市區短距離規劃

## 使用技巧

### 技巧 1：快速新增多個地點
```
1. 輸入地點名稱
2. 按 Enter（不用點按鈕）
3. 立即輸入下一個地點
4. 再按 Enter
5. 重複步驟快速新增

```
1. 開始輸入地點名稱

## 常見問題

### Q: 搜尋不到地點？
**A:** 
- 確認輸入的是台灣的地點
- 嘗試更具體的名稱（例如：「台北101」而不是「101」）
- 檢查網路連線

### Q: 交通工具切換後路線沒變？
**A:** 
- 確認已新增至少 2 個路線點
- 某些地點可能不支援特定交通模式
- 嘗試重新計算路線

### Q: 自動完成沒有出現？
**A:** 
- 確認 Google Maps API Key 已正確設定
- 確認已啟用 Places API
- 檢查瀏覽器 Console 是否有錯誤
