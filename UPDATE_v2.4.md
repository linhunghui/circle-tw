## 🚀 新功能

### 1. **白牌機車專用路線規劃** 🏍️

針對台灣白牌機車（普通重機，排氣量 250cc 以下）的路線限制，新增專用選項。

#### 交通工具選項更新：
- 🚗 **汽車** - 可使用所有道路
- 🏍️ **白牌機車（普通重機）** - 避開國道、快速道路等限制路段
- 🏍️ **紅黃牌機車（大型重機）** - 可使用國道（550cc 以上）
- 🚴 **自行車**
- 🚶 **步行**

#### 白牌機車路線特色：
- ✅ **避開國道**（國道 1 號、3 號、5 號等）
- ✅ **避開快速道路**（台 61、台 62、台 64 等）
- ✅ **避開收費道路**
- ✅ **只規劃一般道路**（省道、縣道、鄉道）
- ✅ **符合台灣交通法規**

#### 技術實作：
```javascript
// 白牌機車設定
travelMode: google.maps.TravelMode.DRIVING
avoidHighways: true  // 避開高速公路
avoidTolls: true     // 避開收費道路
```

### 2. **每日行程顯示騎行公里數** 📊

規劃每日行程時，自動計算每天的騎行距離和時間。

#### 顯示內容：
- 📍 **起點和終點**
- 🛣️ **當天騎行距離**（公里）
- ⏱️ **預估騎行時間**（小時）
- 📊 **總計統計**：
  - 總距離
  - 總騎行時間
  - 天數
  - 平均每天距離

#### 使用方式：
```
1. 新增多個路線點
2. 點擊「規劃每日行程」
3. 為每一天選擇終點
4. 點擊「確定」
5. 系統自動計算每天的距離和時間
```

## 🔧 修復問題

### 清除路線點後地圖標記殘留

**問題**：點擊「清除路線點」後，地圖上的搜尋結果標記（藍色標記）沒有被清除。

**原因**：`searchResultMarker` 是獨立的全域變數，不在 `markers` 陣列中。

**解決方案**：
```javascript
function clearAllWaypoints() {
    // 清除路線點標記
    markers.forEach(marker => marker.setMap(null));
    markers = [];
    
    // 清除搜尋結果標記
    if (searchResultMarker) {
        searchResultMarker.setMap(null);
        searchResultMarker = null;
    }
    
    // 清除資訊視窗
    if (searchResultInfoWindow) {
        searchResultInfoWindow.close();
        searchResultInfoWindow = null;
    }
    
    // 清除路線
    directionsRenderer.setDirections({ routes: [] });
}
```

## 📊 路線比較

### 台北 → 高雄

| 交通工具 | 距離 | 時間 | 路線 |
|---------|------|------|------|
| 🚗 汽車 | ~350 km | ~4 小時 | 國道 1 號或 3 號 |
| 🏍️ 白牌機車 | ~420 km | ~7 小時 | 台 1 線、台 9 線等省道 |
| 🏍️ 紅黃牌機車 | ~350 km | ~4.5 小時 | 國道 1 號或 3 號 |

### 差異說明：
- **白牌機車**：距離較長，但風景更好，可以停靠更多景點
- **紅黃牌機車**：距離較短，時間較快，但需要 550cc 以上才能上國道
- **汽車**：最快速，但無法享受騎車樂趣

## 🔍 技術細節

### Google Maps API 參數

#### avoidHighways
- `true`：避開高速公路（國道）
- `false`：可使用高速公路

#### avoidTolls
- `true`：避開收費道路（包含快速道路）
- `false`：可使用收費道路

### 台灣道路分類

#### 白牌機車可騎：
- ✅ 省道（台 1、台 3、台 9 等）
- ✅ 縣道（縣道 106、縣道 193 等）
- ✅ 鄉道
- ✅ 市區道路

#### 白牌機車禁止：
- ❌ 國道（國 1、國 3、國 5 等）
- ❌ 快速道路（台 61、台 62、台 64 等）
- ❌ 部分高架道路

1. **東部路線**：風景優美但山路較多，建議每天 100-150 km

- ⚠️ 避免夜間騎乘
